## 🚀 What is Download Hub?

Download Hub is a **fast, clean, browser-based game hub** that lets you download and play HTML game files directly — no accounts, no installs, no paywalls. Everything runs in your browser, making it perfect for **Chromebooks, school computers, and any device with a browser**.

All files are hosted directly in this repository and served through GitHub Pages, so downloads are fast, reliable, and always available.

## 🍴 Forking This Project

Want to run your own version of Download Hub? Fork it in one click:

1. Click **Fork** at the top right of this page
2. Go to your fork's **Settings → Pages** and enable GitHub Pages on the `main` branch
3. Your site will be live at `https://yourusername.github.io/Download-Hub/`
4. Visit `your-site/admin.html` to manage your downloads through the built-in admin panel

> ℹ️ Forks share the same Supabase stats database by default. Set up your own free Supabase project if you want separate tracking — instructions are in the admin panel.

## 🛠️ How It Works

The site is **100% static** — no server, no backend (except optional Supabase for stats).

```
Download-Hub/
├── index.html        ← Main site
├── admin.html        ← Password-protected admin panel
├── changelog.html    ← Timeline of updates
├── cards.json        ← All download card data
├── changelog.json    ← Changelog entries
├── config.json       ← Banner settings + Supabase config
├── logo.png          ← Site logo
└── [game files]      ← The actual downloads (.zip, .html)
```

**Adding a new download:**
1. Upload the file to the repo
2. Log into `admin.html` (default password: `DownloadHub2026`)
3. Click **+ Add Card**, fill in the details, hit **🚀 Publish**
4. Done — live in ~30 seconds
